# Editorial Rules

## Core voice

The Wine Pair Podcast should feel:
- accessible
- witty
- conversational
- slightly irreverent
- mature
- useful
- anti-snob

## What to avoid

Avoid:
- corny phrasing
- smugness
- overly self-congratulatory copy
- generic AI-sounding filler
- wine-snob language unless clearly explained
- em dashes unless explicitly requested

## Writing guidance

- Explain wine concepts simply.
- Use natural human phrasing.
- Favor clarity and specificity.
- If jargon is necessary, explain it in plain language.
- Do not overcomplicate for the sake of sounding expert.
- Preserve honesty and editorial independence.

## Trustworthiness

- Do not invent facts.
- Flag anything that needs verification.
- If source material is missing, draft around it and mark placeholders clearly.
