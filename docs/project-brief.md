# Project Brief

## What this project is

A local-first podcast operating system for The Wine Pair Podcast.

The goal is not to create a flashy autonomous agent. The goal is to create a durable, searchable, reusable workflow system that reduces repeated work and preserves podcast context over time.

## Why this exists

Current pain points:
- AI chat memory is limited and drifts
- repeated uploads of the same files are annoying
- podcast work repeats every week
- context is spread across notes, transcripts, Google Drive docs, and chats
- first drafts and file gathering take too much time

## What the system should help with

High-value recurring work:
- research retrieval
- research brief generation
- episode drafting support
- transcript to show-notes packaging
- SEO copy
- chapter marker suggestions
- BlueSky drafts
- Instagram caption drafts
- archive lookup
- listener / guest / sponsor reply drafting

Lower-value or less-automatable work:
- final recording
- final audio edit judgment
- final cover art judgment
- final strategy decisions

## v1 outcome

A usable local system that:
- ingests 1–3 sample episodes
- stores key materials in a clean archive
- supports archive search
- creates a research brief
- creates an episode pack
- reduces repeated file uploads
