# The Wine Pair Podcast – Podcast OS Bundle for Claude Code

This bundle gives Claude Code a practical starting point for building a local-first podcast operating system on your existing home computer.

## What this is

A set of operating documents, starter templates, schema files, and setup notes so Claude Code can:

- build a durable archive
- reduce repeated uploads
- search past work
- retrieve files from synced Google Drive folders
- generate repeatable first drafts for recurring podcast tasks
- save outputs back into a clean local system

## What to do first

1. Put this whole folder somewhere permanent on your home computer.
2. Open this folder in Claude Code.
3. Ask Claude Code to read:
   - `CLAUDE.md`
   - `README.md`
   - everything in `docs/`
4. Then paste the text from `KICKOFF_PROMPT.md` into Claude Code.

## Suggested repo name

`podcast-os`

## What Claude Code should build first

- local folder structure
- SQLite database and init flow
- ingest for 1–3 sample episodes
- archive search
- research brief workflow
- episode pack workflow
- Google Drive-aware file discovery

## Human note

You do not need to know software architecture to use this. The documents are written so Claude Code should figure out implementation details where practical and keep the system understandable.
